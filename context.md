# Project Context: Rahul-Shilpa Anniversary Website

## Tech Stack
- React 18 + TypeScript + Vite
- Tailwind CSS + PostCSS
- Framer Motion (animations)
- Lucide React (icons)

## Couple Details
- Names: Rahul & Shilpa
- Language: Marathi (with English fallback)
- Theme: Romantic, floral, elegant

## Design System
- Primary: Rose/Pink tones
- Secondary: Gold accents
- Font: Marathi script + English fallback
- Background: Floral patterns, soft gradients

## Components (22 total)
- HeroSection, MemoryGallery, TogetherCounter, FirstYearStory
- AnniversaryStats, LittleMoments, MarathiLoveLetter
- SurpriseEnvelope, MusicControl, FloralCorner, MogaraGarland
- + 12 other decorative/functional components

## File Structure
- public/images/ : Couple photos
- public/audio/ : Background music
- src/components/ : All React components
- src/components/hooks/ : Custom hooks (useAudio)
- src/coupleConfig.ts : Couple data configuration
